HOWTO BUILD W_SCAN
==================

ENTER THIS DIRECTORY..

## DOWNLOAD SOURCES ##
```linux shell
$ wget http://wirbel.htpc-forum.de/w_scan/w_scan-20110329.tar.bz2
```

## BUILD W_SCAN ##
```linux shell
$ tar xf w_scan-20110329.tar.bz2
$ ./build_wscan.sh
```
the compiled w_scan binary will then be copied to /res/raw/
